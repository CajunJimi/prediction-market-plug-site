Calibration Curve
What is a calibration curve? For each 5% confidence bucket (e.g. "65–70% confidence"), we measure what fraction of signals actually moved in the predicted direction. A perfectly calibrated model follows the diagonal. Dots above the diagonal are over-performing; dots below are over-confident. Dot size = number of samples.

Why Calibration Matters

Most ML models are either over-confident (say 90%, win 60%) or under-confident. A calibrated model is essential for Kelly-optimal position sizing — you can trust the confidence score as a true probability and size positions accordingly.

Our SignalScoringService updates weights daily as markets resolve. With 3.75M+ labelled signals, the model has enough data to learn true probabilities for each alert type independently.
